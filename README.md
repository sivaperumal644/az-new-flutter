# az-news flutter app

## Description

It is a news app that fetches the news from an api for different fields of news in a tablayout.

## Components

1. Written in dart.
2. Uses newsapi.org JSON API.
3. Made with flutter.

## Features 

- Multiple news categories.
- article previews
